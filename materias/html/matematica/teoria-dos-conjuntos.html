1. Introdução 🎀

A Teoria dos Conjuntos é um ramo fundamental da matemática que estuda coleções de objetos, chamados de elementos. Este guia oferece um panorama detalhado dos principais conceitos, operações e propriedades dos conjuntos, essenciais para o estudo de matemática em vestibulares.

2. Definição e Representação 🎀

Um conjunto é uma coleção de objetos bem definidos, chamados de elementos. Existem várias maneiras de representar conjuntos:

  1. Representação por Extensão: Listamos os elementos entre chaves. Exemplo: A = {1, 2, 3}.

  2. Representação por Compreensão: Descrevemos os elementos por uma propriedade. Exemplo: B = {x | x é par}.

Os diagramas de Venn são uma ferramenta visual útil para representar conjuntos e suas relações.

3. Operações entre Conjuntos 🎀

As operações básicas entre conjuntos incluem união, interseção, diferença e complemento:

  1. União (A ∪ B): Contém todos os elementos de A e B.

  2. Interseção (A ∩ B): Contém apenas os elementos comuns a A e B.

  3. Diferença (A - B): Contém os elementos de A que não estão em B.

  4. Complemento (Aᶜ): Contém todos os elementos que não estão em A.

4. Propriedades 🎀

As propriedades da união e interseção incluem comutatividade, associatividade, distributividade e leis de identidade e complemento:

  1. Comutatividade: A ∪ B = B ∪ A e A ∩ B = B ∩ A.

  2. Associatividade: (A ∪ B) ∪ C = A ∪ (B ∪ C) e (A ∩ B) ∩ C = A ∩ (B ∩ C).

  3. Distributividade: A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C) e A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C).

5. Fórmulas 🎀

Algumas fórmulas úteis para resolver problemas de conjuntos:

  1. Princípio da Inclusão-Exclusão: |A ∪ B| = |A| + |B| - |A ∩ B|.

  2. Complemento: |Aᶜ| = |U| - |A|.

  3. Conjuntos Disjuntos: |A ∪ B| = |A| + |B|.
